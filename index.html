Usage Instructions

1. Загрузите изображение, нажав на кнопку "Выбрать файл".
2. Используйте ползунки для настройки обрезки изображения и размеров квадрата.
3. Настройте порог и инвертируйте изображение, если необходимо.
4. Используйте кнопку "Рисовать", чтобы закрасить области, которые не должны учитываться в расчете площади.
5. Нажмите "Очистить маску", чтобы удалить все закрашенные области.
6. Результат площади отображается под вторым изображением.

1. Upload an image by clicking the "Choose File" button.
2. Use the sliders to adjust the image cropping and square dimensions.
3. Set the threshold and invert the image if necessary.
4. Use the "Draw" button to paint areas that should not be included in the area calculation.
5. Click "Clear Mask" to remove all painted areas.
6. The area result is displayed below the second image.
